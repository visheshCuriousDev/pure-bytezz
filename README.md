# Pure Bitezz Bakery Website

A modern, mobile-friendly one-page website for Pure Bitezz - an Indian bakery specializing in both traditional and latest Indian snacks.

## Features

- **Mobile-First Design**: Fully responsive layout that works perfectly on all devices
- **Modern UI/UX**: Clean design with brown and yellow color scheme (eye-friendly)
- **Tailwind CSS**: Utility-first CSS framework for rapid development
- **Interactive Elements**:
  - Mobile hamburger menu with smooth animations
  - Smooth scrolling navigation
  - Interactive product cards with hover effects
  - Touch-friendly buttons and links
- **Complete Sections**:
  - Hero section with bakery introduction
  - Products section with "Coming Soon" banner
  - About section with bakery philosophy
  - Contact section with WhatsApp/Instagram placeholders
  - Professional footer

## Technology Stack

- **HTML5**: Semantic markup
- **Tailwind CSS**: For responsive styling
- **JavaScript**: For interactive features
- **Font Awesome**: For icons
- **Google Fonts**: Poppins and Playfair Display

## Mobile Optimization

- Touch-friendly buttons (minimum 44x44px)
- Responsive typography
- Mobile hamburger menu
- Optimized images and icons
- Fast loading with CDN resources

## How to Use

1. Open `index.html` in any modern web browser
2. The website is fully functional with no build process required
3. All resources are loaded via CDN

## Customization

To update the website:

1. **Logo**: Replace the placeholder logo in the navigation
2. **Colors**: Update the custom CSS variables in the `<style>` section
3. **Content**: Modify the HTML content in respective sections
4. **Contact Info**: Update the email and social media links when available
5. **Products**: Update the product lists when ready
